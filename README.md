# MS-vibecoding

Microsoft Vibecoding Demo with Software Finland

## What is Vibecoding?

"Vibecoding" (coined by Andrej Karpathy) is AI-assisted development where you describe your intent in natural language and GitHub Copilot writes the code. This demo was built live to showcase that workflow.

## Demo App

A polished **task management** web app built with React + TypeScript + Vite, featuring:

- ✅ Add, complete, and delete tasks
- 🏷️ Category filtering (Work, Learning, Personal)
- 📊 Live progress tracking
- 💅 Microsoft Fluent-inspired design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Build

```bash
npm run build
```

## Tech Stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) – fast dev/build tooling
- [GitHub Copilot](https://github.com/features/copilot) – the vibecoding engine 🚀
